# Brief do Sistema
## Objetivo
Desenvolver um sistema para detectar anomalias em transações financeiras de despesa pública utilizando uma abordagem baseada em grafos heterogêneos.

## Stack Tecnológica
- **Backend:** Python + Flask
- **Banco de Dados:** Grafo Heterogêneo (NetworkX)
- **Análise de Dados:** Pandas, NumPy, Scikit-learn
- **Infraestrutura:** Docker, GitHub Actions (CI/CD)

## Estrutura do Projeto
- **Backend API** para manipulação dos dados.
- **Análise de Redes** para detectar anomalias.
- **Frontend** para visualização dos dados e anomalias detectadas.
