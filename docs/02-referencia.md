# Como usar a pasta _referencia/
- Use somente como leitura para consultar os dados e padrões do sistema.
- Não editar nada dentro de `_referencia/` para garantir que a referência original seja preservada.
- Recriar/adaptar as estruturas de dados no novo projeto com base na referência.

Referências de dados:
- Empenhos
- Fornecedores
- Órgãos
